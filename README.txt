Para mover o guincho, use as teclas "a" e "d"

Para rotacionar o braço principal (conectado ao guincho), use
as teclas "q" e "e"

Para rotacionar o braço secundário (conectado ao principal), use
as teclas "r" e "f"

Para regular o tamanho da corda que segura o objeto, use "r" e "f"
